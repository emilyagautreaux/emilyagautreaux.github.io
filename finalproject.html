CSC106 Final Project

My partner, Kaleigh Archambault and I developed an educational app for children to practice drawing the upper and lower case letters of the alphabet. The user interface was designed to be simplistic, easy to follow, and understand. It includes features such as a score tracker which increases as the user practices drawimg more letters and allows the user to change the color of the tracing tool. These features are meant to enhace the app by making it more entertaining for children and encouraging creativity.
